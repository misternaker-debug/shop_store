# shop_store
*Это учебный проект.*
Shop Agent — это ассистент на основе больших языковых моделей (LLM), который помогает пользователям находить товары в каталоге, анализировать их преимущества и недостатки, а также управлять корзиной покупок. Проект демонстрирует три подхода к построению агентных систем в сфере e‑commerce: от простого вызова инструментов до многоагентной архитектуры с элементами персонализации.

Агент работает с реальной базой данных (PostgreSQL/SQLite), хранит историю предпочтений пользователя и умеет давать рекомендации с честным разбором плюсов и минусов товаров.

Вместо статического каталога и примитивных фильтров мы создаём интеллектуального консультанта, который:

понимает естественный язык пользователя;

самостоятельно решает, какие инструменты вызвать;

запоминает личные предпочтения (бренд, ценовой лимит, категорию);

может делегировать подзадачи специализированным агентам и синтезировать итоговый ответ.

Это не просто чат-бот, а мультиагентная система, где каждый агент отвечает за свою часть процесса: поиск, анализ, ранжирование и обратную связь.
